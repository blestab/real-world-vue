# real-world-vue

## Project setup
```
yarn install
```

### Start fake API server for events data in a seperate CLI window
```
json-server --watch db.json
```

### Compiles and hot-reloads for development
```
yarn serve
```

### Compiles and minifies for production
```
yarn build
```

### Lints and fixes files
```
yarn lint
```
